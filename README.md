# Editor Bidang dan Atribut

Toolkit digitasi kadastral untuk QGIS — alat geometri, atribut, kadastral, dan topologi dalam satu bundel.

## Fitur

### Geometri
- **Explode Parts** — Memecah multipart geometry menjadi single part
- **Auto Close Polygon** — Menutup polygon secara otomatis
- **Pemecahan Bidang** — Memecah bidang tanah menjadi beberapa bagian

### Atribut
- **Swap Attributes** — Menukar nilai antar atribut
- **Copy Attributes** — Menyalin atribut antar fitur
- **Quick Field Fill** — Mengisi field secara cepat
- **Auto Suffix Nomor** — Menambahkan suffix nomor otomatis
- **Batch Rename** — Mengganti nama atribut secara massal

### Kadastral
- **Hitung Luas** — Menghitung luas bidang tanah
- **Validasi** — Validasi dan highlight bidang bermasalah
- **Ekspor ke Excel** — Mengekspor data atribut ke format Excel

### Topologi
- **Topology Checker** — Pengecekan topologi dengan auto-fix

## Kompatibilitas
- QGIS 3.0 atau lebih baru
- Layer PostGIS, Shapefile, dan GeoPackage

## Instalasi

1. Download file `.zip` dari halaman [Releases](https://github.com/fajarSNI/editor-bidang-dan-atribut/releases)
2. Buka QGIS → **Plugins** → **Manage and Install Plugins**
3. Pilih tab **Install from ZIP**
4. Pilih file `.zip` yang sudah didownload
5. Klik **Install Plugin**
